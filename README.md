# Weap UI

- 原生微信小程序UI库

## 说明

- 该小程序的工具均是基于原生小程序开发的，无需第三方学习成本

## 小程序扫码预览

![](https://github.com/laterly/weap-ui/blob/master/screenshot.jpg)

### 组件说明

- 每个组件都是独立的，无互相依赖关系，如仅仅需要某个功能组件，只需要下载components里的某个组件
- 组件的使用可以看pages下对应的页面使用例子

### 通过npm构建

```cmd
npm install weap-ui
```
- 组件使用
```
{
  "usingComponents": {
    "popup":"weap-ui/popup/popup"
  }
}
```
